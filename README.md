# smart-billing
Sistema de gestão de cobrança desenvolvido em Python.

## Sobre o projeto

O SmartBilling tem como objetivo automatizar o processo de criação e controle de cobrança para pequenas empresas.
O projeto será desemvolvido aplicando conceitos de desenvolvimento de software, organização de código e boas práticas utilizadas no mercado.

## Tecnologias utilizadas

-Python
-Git
-GitHub

## Objetivo de aprendizado

Duante o desenvolvimento serão aplicados:

- Estruturas de dados em Python;
- Funções;
- Organização de projetos;
- Organização de arquivos;
- Manipulação de arquivos;
- Tratamento de erros;
- Boas práticas de desenvolvimento.
